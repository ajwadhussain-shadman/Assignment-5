Ans of the Q-1:
The main difference between var,let and const is in how they declare, assign or re-assign ,and their scope.
1.Var: Its the old way of declaring variables , used before JS ES6.It is function or global scoped .It can re declare and assigned the value .
2.Let: Let is like an upgrade version of var, was introduced in ES6.The main difference of it with var is is block scoped and can not be re declare in same scope, but it can be reassigned.
3.Const:Const is used to give a variable fixed or unchangeable value . Its block scoped, its value can not be redeclared or reassigned and must be assigned when we declare it .
So thats the main difference between them .
